7 Views

State Management

App.tsx (root) owns all state via hooks: useAppShellState(), useWS(), useLeads(), useDueFollowups(), useGraphStats(). Props are drilled into all 7 views. Custom window.dispatchEvent() for inter-component comms. No React Router -- a simple string state selects the current view.

Scoring Engine

Ranking & Agents

Multi-criteria lead scoring with 6 weighted dimensions and feedback learning.

Weights shown are for semantic-enabled mode (total=100). When semantic is unavailable, weights redistribute: Stack 27%, Evidence 20%, Role 18%, Seniority 20%, Logistics 15% (total=100). Score caps: wrong-field max 15, seniority 30-48, stack 42-52. Feedback learning adjusts weights by +/- delta based on user thumbs-up/down.
